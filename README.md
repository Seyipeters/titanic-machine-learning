# Titanic Machine Learning

This project contains Titanic dataset preprocessing for machine learning.

Files:
- `train.csv` - training data
- `test.csv` - test data
- `gender_submission.csv` - sample submission
- `Titanic_Preprocessing.ipynb` - notebook for exploratory data preprocessing

## Setup

1. Install required Python packages.
2. Open `Titanic_Preprocessing.ipynb` in Jupyter or VS Code.
3. Run preprocessing and model training steps in the notebook.

## Notes

- The notebook focuses on data cleaning, feature engineering, and preparing the Titanic dataset for classification.
- Add a `requirements.txt` if you want reproducible package installs.
